# Unstop Ui Assessment




## Deployment

    unstop-fd.vercel.app



## Run Locally

Clone the project

```bash
  git clone https://github.com/abhisheksahu2001/Unstop_FD.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
